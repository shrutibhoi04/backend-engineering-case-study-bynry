## 📌 Project Overview

This case study implements backend solutions for **StockFlow**, a B2B inventory management system.
The system supports multi-warehouse inventory tracking, supplier management, and real-time low stock alerts.

---

## 🚀 Features Implemented

### ✔ Part 1: Code Review & Debugging

* Identified critical backend issues in API design
* Fixed data consistency problems using transactions
* Implemented validation and error handling

---

### ✔ Part 2: Database Design

* Designed scalable schema for:

  * Companies
  * Warehouses
  * Products
  * Inventory
  * Suppliers
  * Bundles
* Added constraints, indexing, and normalization

---

### ✔ Part 3: API Implementation

* Developed low-stock alert system
* Integrated sales-based stock prediction
* Provided supplier details for reordering
